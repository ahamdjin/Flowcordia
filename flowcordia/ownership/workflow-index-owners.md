# Workflow index failure owners

| Failure | Owner |
| --- | --- |
| project/repository connection invalid | webapp scope resolver |
| GitHub installation access | GitHub App operator |
| tree truncated or invalid | GitHub discovery boundary |
| workflow validation | workflow author and canonical contract |
| sync lease/generation conflict | workflow-index repository |
| push replay mismatch | security operator |
| background processing age | Flowcordia operations worker |
| exact source mismatch | Studio query boundary |
| canvas presentation bug | Workflow Studio UI |
