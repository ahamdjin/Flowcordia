# Workflow Studio visible states

- not connected;
- not indexed;
- pending;
- running;
- idle/current;
- stale/update pending;
- failed while retaining last good catalog;
- empty repository;
- valid workflow;
- invalid workflow;
- exact-source mismatch;
- read-only user;
- write user synchronizing.

No state is replaced with mock content or a success-looking placeholder.
