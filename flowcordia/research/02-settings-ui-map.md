# Settings UI map

Organization settings has:

- parent settings route
- side menu component
- child pages rendered in the main body

The side menu changes based on product mode, feature flags, plugin state, billing state, and admin access.

Flowcordia order for future work:

1. add a direct child route
2. test the direct route URL
3. then add a side menu link

Do not use the side menu as the first place to test new pages.
