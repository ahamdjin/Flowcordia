# Workflow Studio useful state

The page is useful before editing because it answers, from proven repository data:

- Which canonical workflows exist?
- Which ones are valid?
- Which commit is being shown?
- What nodes and edges make up the workflow?
- Which operations, runtime hints, credentials, and code references are involved?
- Is the index current, pending, or failed?

Every answer is connected to the actual production repository rather than seeded demonstration data.
