# Workflow index source-of-truth rules

- GitHub commit content is authoritative for workflow definitions.
- The durable index is authoritative for synchronization operations and repository-scoped discovery projection.
- Existing Trigger.dev records are authoritative for organization, project, installation, repository, and branch connection identity.
- GitHub remains authoritative for branch movement.
- The browser is authoritative for no security or repository coordinates.
- The canvas is a projection, never a source of truth in this read-only slice.
